AUDIT - AUDIT SI

Intervalle Technologies offers its customers its "Information System Security Audit" service in accordance with international standards (ISO 2700 X) in force.

INTRUSION TESTS

Whether internal or external, intrusion tests are intended to simulate the behavior of a hacker or malicious user.

The Methodology Intervallr involves analyzing technical vulnerabilities through a wide range of internal tests and external as well as the search for security vulnerabilities and elements that could compromise the information system (example, poor implementation or configuration…).

Our auditors subsequently assess the risks incurred to propose appropriate corrective actions.

AUDIT OF WEB AND MOBILE APPLICATIONS

Before any Web application is implemented in Intranet or Extranet, a Web Security Audit is required to protect itself from possible computer attacks.

In this context, Intervalle Technologies offers its Audit Applications Web service in line with the recommendations of the Free Web Application Security Security Project (OWASP TOP 10), Web application specialist.

OUR METHODOLOGY

Our application audit methodology includes the search for faults in the design and/or development of the application in question.

Audit and consulting services are developed to enable our clients to identify, manage and control the risks associated with the security of information systems.
